# webpack_react_tur
webpack react start project

### build steps defined in 'package.json'
``` 
"scripts": {
    "dev": "webpack-dev-server --config webpack.config.dev.js",
    "prod": "webpack --config webpack.config.prod.js"
  },
```

#### for developing
```
$ npm run dev
```

#### for production build
```
$ npm run prod
```
